# Net-Flow Site

High-end brand site for Net-Flow built with Next.js, React, TypeScript and Tailwind CSS.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React

## Run locally

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```

## Project notes

- The site is a premium one-page company website focused on web, e-commerce, integrations and NetFlow-CMS.
- Brand assets are stored in `public/brand`.
- SEO metadata, `robots.txt` and `sitemap.xml` are generated from the App Router.
